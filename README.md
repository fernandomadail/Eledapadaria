# Padaria Compra Facil

Aplicativo estatico pronto para deploy na Vercel.

## Como publicar

1. Envie esta pasta para um repositorio Git.
2. Na Vercel, crie um novo projeto apontando para o repositorio.
3. Use as configuracoes padrao. Nao ha comando de build.
4. O diretorio de saida e a propria raiz do projeto.

Arquivos principais:

- `index.html`
- `styles.css`
- `app.js`
- `vercel.json`
